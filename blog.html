1. What are the different ways to select an element in the DOM?

In the DOM (Document Object Model) using several methods to select an element. Here are the most common ways: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll().

2. What is the difference between innerHTML, innerText, and textContent?

1.innerHTML (Includes HTML Elements): sets the HTML content of an element, including nested elements.

2. innerText (Visible Text Only): sets the visible text of an element (excluding hidden elements).

3. textContent (All Text, No Formatting): sets all text content, including hidden elements, without parsing HTML.

3. What is event delegation in the DOM?

Event delegation is a technique where a parent element manages events for its child elements through event bubbling. It is especially useful for handling dynamically added elements efficiently.

4. What is event bubbling in the DOM?

Event Bubbling is a process in the DOM where an event starts from the target element (where it was triggered) and propagates up through its ancestors to the root of the document.

5. How do you create, add, and remove elements using JavaScript?

In JavaScript create an element by using document.createElement(), add it using appendChild() or insertBefore(), and remove it using removeChild() or remove().
